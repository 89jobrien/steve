# HTML Rules

## Formatting

- 2-space indentation
- Use semantic HTML5 elements
- Include `lang` attribute on `<html>`

## Accessibility

- All images need `alt` attributes
- Form inputs need associated labels
- Use ARIA attributes when semantic HTML isn't sufficient
